## 游戏背景

一场瘟疫危机席卷全球！你将扮演医疗指挥官，派出科学家、医生、药剂师等力量，与时间赛跑，对抗新冠病毒、肺炎、甲流等各种传染病。每一轮攻防都需要你运用医学知识正确回答治疗方案和症状，才能夺回主动权。

## 游戏规则

### 基本流程

```
对手出牌攻击 → 你防御应对 → 回答问题 → 你出牌攻击 → 对手防御应对 → 循环往复
```

### 详细规则

#### 1. 开局发牌
- 双方各抽 **5 张**起手牌
- 玩家为先手，首先攻击

#### 2. 攻击阶段（出疾病卡）
- 攻击方从手牌中选择 **1 张疾病卡** 打出
- 该疾病的 **等级** 即为本轮对手需要达到的应对门槛
- 打出后，该卡从手牌中移除

#### 3. 防御阶段（出治疗卡）
- 防御方从手牌中选择 **任意张** 治疗卡 和/或 时间卡
- 所选卡牌的 **等级之和** 必须 ≥ 攻击方疾病卡的等级
- 确认出牌后，所用卡牌从手牌中移除

#### 4. 疫苗特殊效果
- 若防御卡组合中 **包含疫苗**，则本轮治疗 **总等级 ×2**
- 例如：科学家(Lv.7) + 疫苗 → 有效等级 = 7 × 2 = 14
- 疫苗使用后同样从手牌中移除

#### 5. 知识问答
- 防御成功后，防御方需回答所应对疾病的知识问题：
  - **治疗方法**：从 4 个选项中选 **1 个**正确答案
  - **症状**：从 6 个选项中选 **2 个**正确答案
- ✅ 全部正确 → 防御方转为攻击方，轮到你出牌
- ❌ 任一错误 → 防御失败，对方继续攻击

> 💡 电脑防御时自动给出正确答案，无需担心电脑答错。

#### 6. 抽牌机制
- 若轮到某方攻击但手牌中 **没有疾病卡**，从牌堆抽牌直到抽到疾病卡
- 若轮到某方防御但手牌 **无法凑够等级**，从牌堆抽牌直到能凑够
- 牌堆耗尽时仍无法出牌 → **该方判负**

#### 7. 胜利条件
- 🏆 **先出完所有手牌** 的一方获胜
- 💀 牌堆耗尽后，**最后一轮无法出牌** 的一方判负

## 难度说明

游戏提供三种难度级别，在开局时选择：

### 😊 简单
- 电脑随机选择疾病卡攻击（不挑最高等级）
- 电脑防御时不追求最优组合，可能浪费卡牌
- 电脑使用疫苗概率较低（~30%）
- **适合**：新手入门，熟悉规则

### 🤔 普通
- 电脑优先出最高等级疾病卡
- 电脑防御时追求最优策略（最少卡牌凑够等级）
- 电脑大概率使用疫苗（~70%）
- **适合**：有一定经验的玩家

### 😈 困难
- 电脑始终出最高等级疾病卡
- 电脑防御时做到精确最优
- 电脑必用疫苗（只要有就用在最优时机）
- **适合**：挑战自我，追求极限

游戏中需要回答疾病相关的医学知识。以下是各疾病的正确答案：

|------|----------|

## 策略提示

### 🎯 基本原则
1. **卡牌管理**：不要一味出最强牌，留几张好牌用于防御
2. **疫苗时机**：疫苗能翻倍，留着应对高等级疾病（如新冠病毒 Lv.10）
3. **时间卡价值**：时间卡等级低但数量多，搭配治疗卡凑等级

### 🧠 进阶技巧
4. **计算消耗**：出牌前估算——用 2 张小卡还是 1 张大卡更划算？
5. **抽牌风险**：牌堆越少，抽不到关键牌的风险越大
6. **知识准备**：记住上方的疾病知识点，问答环节不翻车

### 🆚 对抗不同难度
- **打简单电脑**：电脑会浪费卡牌，稳定发挥就能赢
- **打普通电脑**：需要合理配卡，答对问题是关键
- **打困难电脑**：电脑每次最优出牌，你需要更精准的卡牌规划和知识储备
